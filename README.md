# PrimbonJodoh
Web app metode penanggalan dan perhitungan hari lahir dalam budaya Jawa yang digunakan untuk memprediksi, kecocokan jodoh seseorang. Weton diperoleh dengan menggabungkan hari lahir (Senin-Minggu) dengan hari pasaran Jawa (Legi, Pahing, Pon, Wage, Kliwon). 
https://arsssyaddhani.github.io/PrimbonJodoh/
